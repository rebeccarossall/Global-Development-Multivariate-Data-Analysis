# Global Development Analysis Using Principal Component Analysis

**Awarded 100%** in a third-year University of Nottingham statistical analysis project.

This project analyses GDP per capita, life expectancy and population data from 141 countries spanning 55 years. Using exploratory data analysis, Principal Component Analysis (PCA) and classification methods in R, it identifies long-term global development patterns and explores relationships between economic and demographic variables.
	
## Skills Demonstrated

- Exploratory Data Analysis (EDA)
- Principal Component Analysis (PCA)
- Classification Methods
- Statistical Modelling
- Data Visualisation
- R Programming
