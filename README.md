# Global Development Multivariate Data Analysis

This project analyses GDP per capita, life expectancy, and population data from 141 countries spanning 55 years. The analysis combines exploratory data analysis (EDA), multivariate statistical techniques, and predictive modelling in R to identify long-term global development patterns and investigate relationships between economic and demographic variables across countries.

This project is based on a final-year university statistical analysis assignment that achieved a mark of **100%**.
	
## Skills Demonstrated

- Exploratory Data Analysis (EDA)
- Principal Component Analysis (PCA)
- Canonical Correlation Analysis (CCA)
- Multidimensional Scaling (MDS)
- Linear Discriminant Analysis (LDA)
- Hierarchical Clustering
- K-Means Clustering
- Regression Modelling (OLS, Principal Component Regression & Ridge Regression)
- Data Visualisation
- R Programming
- Statistical Reporting

## Key Highlights

- Conducted exploratory data analysis (EDA) using time series visualisations, summary statistics, histograms, and correlation analysis to investigate global development trends.
- Applied Principal Component Analysis (PCA) to reduce dimensionality and identify the main sources of variation across countries.
- Performed Canonical Correlation Analysis (CCA) to investigate relationships between economic and demographic variables.
- Applied Multidimensional Scaling (MDS), hierarchical clustering, and K-Means clustering to identify groups of countries with similar development characteristics.
- Built and compared Ordinary Least Squares (OLS), Principal Component Regression (PCR), and Ridge Regression models to predict life expectancy.
- Used Linear Discriminant Analysis (LDA) to classify countries based on development characteristics.
